# Tab

Tab component for displaying tabs.

#### Example

<Canvas>
  <Tab
    active={true}
    onClickHandler={() => console.log("Tab clicked")}
    title="Tab 1"
  />
  <Tab
    active={false}
    onClickHandler={() => console.log("Tab clicked")}
    title="Tab 2"
  />
</Canvas>

## Usage

```tsx
import React from "react";
import Tab from "./Tab";

const Example = () => {
  return (
    <div>
      <Tab
        active={true}
        onClickHandler={() => console.log("Tab clicked")}
        title="Tab 1"
      />
      <Tab
        active={false}
        onClickHandler={() => console.log("Tab clicked")}
        title="Tab 2"
      />
    </div>
  );
};

export default Example;
```

#### Arguments

- **active** _`boolean`_ - Indicates whether the tab is currently active or not.
- **onClickHandler**_`() => void`_ - Function to be called when the tab is clicked.
- **title**_`string`_ - The title or label of the tab.
- **disabled** _`boolean`_ - Indicates whether the tab is disabled or not.
