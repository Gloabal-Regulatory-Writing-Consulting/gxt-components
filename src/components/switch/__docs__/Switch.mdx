# Switch Example

This is an example component that showcases the usage of `Switch` components.

## Usage

```tsx

const SwitchExample = () => (
  const [isToggled, setIsToggled] = useState(false);

  const handleToggle = () => {
    setIsToggled(!isToggled);
  };

  return <Switch checked={isToggled} onChange={handleToggle} />
);

export default SwitchExample;
```
