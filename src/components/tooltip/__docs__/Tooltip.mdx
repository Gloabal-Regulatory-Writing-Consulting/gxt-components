# Tooltip Example

This is an example component that showcases the usage of `Tooltip` components.

## Usage

```tsx

const TooltipExample = () => (
  <div style={{ width: "20rem", color: "#414141" }}>
    <Tooltip
      text="I am a tooltip"
      position="top"
      background="222831"
      tooltipTextColor="#fff"
    >
      Hover me
    </Tooltip>
  </div>
);

export default TooltipExample;
```
