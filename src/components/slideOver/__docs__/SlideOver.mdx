#### Arguments for SlideOver

- **isOpen** - A boolean indicating whether the SlideOver is open. Default is false.
- **onClose** - Function called when the overlay is clicked. It receives a VoidFunction for handling the click event on the overlay element.
- **position** - The position of the SlideOver. Default is "right". Options are "left" or "right"
- **overlay** - A boolean indicating whether the SlideOver should have an overlay. Default is true.
- **width** - The width of the SlideOver. Default is "250px".
- **overlayClasses** - The classes of the overlay.
- **contentClasses** - The classes of the content.
- **slideOverClasses** - The classes of the SlideOver.
- **slideOverStyles** - The styles of the SlideOver.
- **overlayStyles** - The styles of the overlay.
- **contentStyles** - The styles of the content.
