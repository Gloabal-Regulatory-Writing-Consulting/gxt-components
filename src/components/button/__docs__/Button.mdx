# Button

Button component with different props.

#### Example

<Canvas of={Button.Primary} />

## Usage

```ts
import { Button } from '@gloabal-regulatory-writing-consulting/gxt-components';;

const ExampleButton = () => {
 const handleClick = (event: React.MouseEvent) => {
    // Handle button click
  };

  return (
    <Button
      title="Button"
      onClick={handleClick}
      disabled={false}
      size="medium"
      variant="primary"
    />
  );
};

export default ExampleButton;
```

#### Arguments

- **title** - Text or node to render inside the button.
- **loading** -  A boolean indicating whether to show a loading spinner in the button.
- **disabled** -  A boolean indicating whether the button should be disabled.
- **classNames** - Additional CSS classes to apply to the button.
- **variant** `"primary" | "secondary" | "negative" | "positive"` - The style variant of the button.
- **onClick** - Function called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
